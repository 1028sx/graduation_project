# 敦煌

一款以敦煌為主題的2D動作遊戲，使用 Godot 4.3.stable.official [77dcf97d8] 開發。

## 操作方式

- AD：移動
- 滑鼠左鍵：普通攻擊
- E：特殊攻擊
- Space：跳躍
- Shift：衝刺
- Q：開啟物品欄
- Esc：暫停選單

## 安裝說明

1. 下載最新版本的遊戲
2. 解壓縮檔案
3. 執行 DunHuang.exe
